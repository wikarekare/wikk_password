wikk_password

  • Gem :: rubygems.org/gems/wikk_password

DESCRIPTION:

Reads/writes password file entries of format user:password and provides tests for password validity. Works with standard unix password types, clear text for testing, and decryptable AES_256_CBC.

FEATURES/PROBLEMS:

*Should have locking around the changes to the password file, but haven't gotten around to the Lockfile gem yet.
